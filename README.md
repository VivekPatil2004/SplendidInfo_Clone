# Splendid Infosystems Clone

This repository contains a frontend clone website for **Splendid Infosystems**, a software company specializing in mobile, web, and cloud software development.

## 🚀 Overview

The website highlights the company's services, vision, and mission. It provides information regarding their unique offerings tailored for aspiring IT professionals:
- **Software Development:** Innovative solutions for Web, Mobile, and Cloud.
- **Internships:** Real-time project experience for freshers.
- **Pre Placement Campaigns:** Foundational training for careers in top tech companies.
- **Logic Building Courses:** Fundamental coding logic to crack interviews and excel in software roles.

## 🛠️ Technologies Used

- **HTML5:** Semantic architecture and structure.
- **CSS3:** Styling, animations, flexbox layouts, and responsive design.
- **Font Awesome:** Scalable vector icons.
- **Google Fonts:** Modern typography styling.

## 📂 Project Structure

- `index.html`: The main landing page document.
- `Style.css` / `Home.css` / `About.css` / `Services.css` / `offer.css` / `Achivement.css` / `Contact.css`: Modular stylesheets for different sections of the website.
- `Images/`: Directory containing all graphical assets, icons, and illustrations.

## 🌟 Key Features

- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
- **Modern UI Elements:** Clean layout with interactive hover states.
- **Floating Action Button:** WhatsApp integration for quick communication.
- **Smooth Navigation:** Sticky header with a hamburger menu for mobile devices.
- **Working Contact Section:** An intuitive contact form and embedded details to reach the organization.

## 💻 Getting Started

To view the website locally:

1. Clone this repository:
   ```bash
   git clone git@github.com:VivekPatil2004/SplendidInfo_Clone.git
   ```
2. Navigate into the cloned directory:
   ```bash
   cd SplendidInfo_Clone
   ```
3. Open `index.html` in your favorite web browser.
4. No build tools (like npm or Node.js) are required to run this static website!
